# Fastaxii Api


## Quick Start

  Clone the repository:

```bash
$ git clone git@github.com:amitdubey8888/fastaxii-api.git
```

  Goto project directory:

```bash
$ cd fastaxii-api
```

  Install dependencies:

```bash
$ npm install
```

  Start the app:

```bash
$ npm start
```

  View the app at: http://localhost:3000
